# Avantex Dashboard — Sistema de Monitoreo de Pauta Digital

Dashboard de campañas Meta Ads + Google Ads para Grupo Avantex.

## Deploy en Render

1. Conecta este repositorio en render.com
2. Configura la variable de entorno: `ANTHROPIC_API_KEY=sk-ant-...`
3. Start command: `node server.js`
